# Abhinav Mantri Portfolio

Static GitHub Pages portfolio for Abhinav Mantri.

## Publish On GitHub Pages

Recommended repository name:

```text
AbhinavMantri.github.io
```

After pushing this folder to that repository, the public portfolio URL will be:

```text
https://AbhinavMantri.github.io
```

## Files

- `index.html` - portfolio content
- `styles.css` - responsive visual design

## Notes

- No build step is required.
- Keep the GitHub repository public.
- In GitHub repository settings, enable Pages from the `main` branch root if it is not enabled automatically.
